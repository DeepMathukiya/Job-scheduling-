Shortest Job First(SJF) Scheduling Algorihtm

"Shortest Job First (SJF) is a scheduling algorithm utilized in computer operating systems. It prioritizes tasks based on their burst time, aiming to execute the shortest job first. If a new task arrives with a shorter burst time than the one currently running, SJF switches to the new task, thus minimizing overall completion time."

void findWT(Process proc[], int n, 
                        int tat[], int wt[]) 
{ 
    for (int i = 0; i < n; i++) 
        wt[i] = tat[i] - proc[i].bt; 
} 

void findavgTime(Process proc[], int n)
